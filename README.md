# The Gemma - Performance Dashboard

Premium performance dashboard for The Gemma vacation rental property.

## Features

- **3 Hero Metrics**: Revenue, Occupancy, ADR (Current Year vs Same Time Last Year)
- **Budget vs Actual Analysis**: High-level performance overview
- **Revenue Trend Chart**: Full-width STLY pacing visualization
- **Detailed Tables**: Monthly budget vs actual breakdown for Revenue, Occupancy, and ADR
- **Budget vs Reforecast**: Q1 actuals + Q2-Q4 reforecast analysis
- **Strategic Insights**: Actionable recommendations

## Data Sources

- Unit night pacing pivot (2026-04-07)
- 2026 Budget vs Reforecast comparison
- STLY (Same Time Last Year) benchmarking

## Key Insights

- Revenue pacing essentially flat vs STLY (-0.2%)
- Occupancy down 17.8% vs STLY (main challenge area)
- ADR up 2.1% vs STLY (pricing power maintained)
- Premium positioning strategy working despite volume challenges

## Design

- Gem-themed premium design with diamond patterns
- Responsive layout for all screen sizes
- Interactive Chart.js visualizations
- Color-coded performance indicators
- Mobile-friendly table layouts
